Dining Services & Kitchen Production Landing Page
Culinary Excellence Rooted in Aloha
📌 Project Overview
This repository contains the source code for the internal landing page for the Dining Services and Kitchen Production department at the Polynesian Cultural Center (PCC). The site is designed to introduce employees to the department's mission, values, and operational standards.

 "Design & Branding Standards
Motto: "Hospitality from the Heart".

Hawaiian Branding: The phrase Mea Hōʻokipa must always include the kahakō over the 'ō' to maintain cultural accuracy.

Typography: Uses Playfair Display for a premium serif look and Lato for clean, readable body text.

Color Palette:

Gold: #B8912A

Warm Brown: #4A2E10

Cream: #FAF6EE

"Technical Stack
Frontend: HTML5, CSS3 (using CSS Variables and Flexbox/Grid), and Vanilla JavaScript.

Interactivity: Includes an IntersectionObserver for fade-in animations on scroll.

Assets:

LOGO.svg: The official Dining Services logo.

hero-bg.mp4: (Planned) Cinematic B-roll captured via Sony A6500.

📂 Folder Structure
/index.html - Main entry point.

/assets/ - Contains SVG logos and compressed web video.

/css/ - (Optional) External stylesheets.
