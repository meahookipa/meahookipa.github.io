How to Update the Website

Step-by-step guide for publishing changes to GitHub — no coding needed.

Step-by-step instructions
Step 1 — Edit your content here
  1. Pick a page from the sidebar on the left (e.g. "Home Page")
  2. Update any text in the boxes — just type like you would in Word
  3. Click Generate Updated Page at the bottom
  4. Click Copy to Clipboard on the code that appears
Step 2 — Paste to GitHub
  1. Go to github.com and sign in to the meahookipa account
  2. Click on the repository meahookipa.github.io
  3. Find the file you want to update (e.g. index.html for Home, or alii-luau/index.html for Ali'i Luau)
  4. Click the pencil icon ✏️ on the top right of the file
  5. Press Ctrl+A (Windows) or Cmd+A (Mac) to select all the existing code
  6. Press Ctrl+V (Windows) or Cmd+V (Mac) to paste your new code
  7. Scroll down and click Commit changes
  8. Click Commit changes again in the popup
  9. Wait about 1 minute — your site is now updated! ✅
Which file = which page?
  1. index.html → Home page
  2. alii-luau/index.html → Ali'i Luau page
  3. gateway-buffet/index.html → Gateway Buffet page
  4. team/index.html → Team / Org Chart page
  5. policies/index.html → Policies & Procedures page
